# Archive Directory

This directory contains historical documentation and temporary files that are no longer actively maintained but are kept for reference.

## Structure

### agent-summaries/
Contains reports and summaries generated during various AI agent development sessions:
- AGENT_ORCHESTRATION_SUMMARY.md - Multi-agent coordination system documentation
- AI_AGENT_MANIFEST.md - AI agent configuration manifest
- BREWING_WORKFLOW_IMPLEMENTATION.md - Brewing workflow feature implementation
- COMPREHENSIVE_BREWING_WORKFLOW_SUMMARY.md - Detailed brewing workflow documentation
- FRONTEND_OVERHAUL_SUMMARY.md - Frontend overhaul project summary
- IMPLEMENTATION_STATUS.md - Historical implementation status tracking
- PHASE_1_COMPLETION_REPORT.md - Phase 1 project completion report
- POST_MIGRATION_SUMMARY.md - Post-migration documentation
- TASK_COMPLETION_REPORT.md - Task completion tracking

### temporary/
Contains temporary files and outdated documentation:
- analysis-notes.txt - Frontend analysis notes
- deploy_parallel_agents.sh - Outdated parallel agent deployment script
- README_UPDATE_TEMPLATE.md - Template for README updates

### Other Files
- MIGRATION_GUIDE.md - Guide for the repository migration
- Project Setup Guide.md - Original project setup documentation

## Usage

These files are maintained for historical reference and should not be actively modified. For current project documentation, see the root-level README.md, ROADMAP.md, and TODO.md files.
