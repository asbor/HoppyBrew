## üìù Recipe Management

### Creating a New Recipe

1. **Navigate to Recipes**
   - Click "Recipes" in sidebar
   - Click "New Recipe" button

2. **Basic Information**
   - Name your recipe
   - Select beer style (BJCP guidelines)
   - Choose batch size
   - Select equipment profile

3. **Add Ingredients**
   - **Fermentables**: Base malts, specialty grains
   - **Hops**: Bittering, flavor, aroma
   - **Yeast**: Select strain and amount
   - **Miscellaneous**: Irish moss, whirlfloc, etc.

4. **Mash Profile**
   - Choose mash schedule
   - Set temperatures and durations
   - Configure efficiency

5. **Fermentation Profile**
   - Primary fermentation temperature
   - Duration and timing
   - Secondary fermentation (optional)

6. **Save Recipe**
   - Click "Save"
   - Recipe added to your library

### Editing Recipes

- Click any recipe to view details
- Click "Edit" to modify
- Use "Clone" to create variations

### Recipe Import/Export

**Import BeerXML**
1. Click "Import" button
2. Select BeerXML file
3. Review imported data
4. Save to library

**Export Recipe**
1. Open recipe
2. Click "Export"
3. Choose format (BeerXML, PDF)
4. Download file

## üç∫ Batch Tracking

### Starting a Batch

1. **Select Recipe**
   - Open recipe details
   - Click "Start Batch"

2. **Configure Batch**
   - Set brew date
   - Adjust volumes if needed
   - Assign equipment

3. **Check Inventory**
   - System checks ingredient availability
   - Allocates ingredients from inventory
   - Warns of shortages

4. **Create Batch**
   - Batch moves to "Planning" status
   - Appears in active batches

### Brew Day Workflow

**Planning Stage**
- Review recipe and ingredients
- Prepare equipment
- Calculate water volumes

**Brewing Stage**
1. Mark batch as "Brewing"
2. Log mash temperatures
3. Record boil timing
4. Log gravity readings (OG)

**Fermentation Stage**
1. Move to "Primary Fermentation"
2. Log fermentation temperature
3. Record gravity readings
4. Monitor progress

**Packaging Stage**
1. Mark batch as "Conditioning" or "Kegged"
2. Log final gravity (FG)
3. Calculate ABV
4. Set packaging date

**Completed**
- Archive finished batch
- Update inventory for consumed ingredients
- Rate and review

### Batch Monitoring

- **Status Dashboard**: View all active batches
- **Temperature Tracking**: Log fermentation temps
- **Gravity Readings**: Track attenuation
- **Notes**: Add observations
- **Photos**: Document progress

## üí° Tips & Best Practices

### Recipe Development

1. **Start Simple**: Begin with proven recipes
2. **Take Notes**: Document all changes
3. **Clone Recipes**: Create variations
4. **Track Results**: Rate and review batches
5. **Learn Styles**: Study BJCP guidelines

### Batch Management

1. **Plan Ahead**: Check inventory before brewing
2. **Log Everything**: Temperature, gravity, notes
3. **Be Consistent**: Use same equipment
4. **Monitor Closely**: Check fermentation daily
5. **Sanitize**: Log cleaning procedures

### Inventory Control

1. **Buy in Bulk**: Save money on common ingredients
2. **Rotate Stock**: Use FIFO (first in, first out)
3. **Store Properly**: Hops in freezer, grain in cool/dry
4. **Track Costs**: Monitor brewing expenses
5. **Set Alerts**: Enable low stock notifications

### Using Calculators

1. **Double Check**: Verify calculations
2. **Save History**: Keep calculation records
3. **Round Appropriately**: Don't over-precision
4. **Understand Formulas**: Know what you're calculating
5. **Use Templates**: Save common calculations

## ‚ùì FAQ

**Q: Can I import recipes from Brewfather?**  
A: Yes, export as BeerXML from Brewfather and import to HoppyBrew.

**Q: How accurate are the calculators?**  
A: Calculators use industry-standard formulas. Always verify with your equipment.

**Q: Can I brew multiple batches simultaneously?**  
A: Yes, track unlimited batches at once.

**Q: Is my data backed up?**  
A: Use the export feature regularly. Set up automated backups.

**Q: Can I use this commercially?**  
A: HoppyBrew is designed for homebrewing. For commercial use, please contact us.
