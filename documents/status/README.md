# Status Log Conventions

- One markdown file per work session: `YYYY-MM-DD.md`.
- Start with an “Overview” section (what changed, biggest risks).
- Append “Next Steps” and “Questions/Blockers”.
- Reference issues/PRs or TODO/Roadmap IDs where possible.

Example skeleton:
```
# 2024-05-31

## Overview
- ...

## Next Steps
- [ ]

## Questions / Blockers
- ...
```
